## 📌 About the Project

The **Interactive Yearbook Viewer** is a dynamic, book-style interface that brings web experiences to life using only HTML, CSS, and vanilla JavaScript. Originally created as a fun visual experiment after the success of a school cafeteria ordering app, it quickly evolved into something far greater.

---

## 🌟 Real-World Impact

This project has had a real and exciting impact:

- ✅ After the great experience with the **cafeteria website** and this **interactive viewer**, I was **personally asked** to develop the **official school website for the yearbook**.
- 👨‍🏫 I received **unbelievably good comments** from professors at **Anáhuac Xalapa** and **Universidades Colón**.
- 🏫 The project was highly praised by my school’s **highest administrators**, and is now in the process of being **implemented**.
- 💡 It became a true example of how creative coding can lead to **real-world invitations and responsibility**.
- 🚀 This is no longer just a project — it's part of the school's digital identity.

## 🚀 Features

- 📖 Realistic **page-flipping animation** for an immersive book experience
- 🧠 **Keyboard shortcuts** (like `1` and `l`) for instant jumps
- 🧩 Programmatically generates **37 front-and-back pages**
- 💻 Fully responsive on mobile and desktop
- 🎯 Includes animated buttons, directional circles, and navigation controls
- ⚙️ Built from scratch using DOM manipulation and vanilla JavaScript

---

## 🛠️ Technologies

- **HTML5** – Semantic structure
- **CSS3** – Responsive layout, transitions, and visual polish
- **JavaScript (ES6)** – Dynamic logic and page generation
- No frameworks, no libraries — just pure creativity and code
